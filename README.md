# Enterprise-Grade RAG + Decision Intelligence Agent

A production-ready, full-stack GenAI SaaS application for intelligent business decision support.

## Architecture

- **Backend**: Python + FastAPI + LangGraph
- **Frontend**: Vite + React + Tailwind CSS
- **AI**: Multi-agent system with hybrid RAG
- **Database**: MongoDB + Redis + FAISS
- **Deployment**: Docker + Railway/Vercel

## Project Structure
```
rag-decision-agent/
├── backend/       # FastAPI backend with LangGraph agents
├── frontend/      # React frontend
├── docker/        # Docker configurations
├── docs/          # Documentation
└── data/          # Local development data
```
